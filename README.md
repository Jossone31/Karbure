# Karbure

Karbure est une application React/Vite pour comparer les prix des carburants des stations-service en France.

## Fonctionnalites

- Geolocalisation des stations proches
- Comparaison des prix par carburant
- Carte interactive
- Statistiques de prix
- Application PWA installable

## Installation

```bash
npm install
```

## Developpement

```bash
npm run dev
```

L'application est disponible par defaut sur `http://localhost:5173`.

## Build

```bash
npm run build
```
