# 🚗 Karbure - Comparateur de Prix Carburant

Karbure est une application mobile progressive (PWA) pour comparer les prix des carburants des stations services en France. Elle utilise les données en temps réel de l'API data.gouv.fr.

## 🌟 Fonctionnalités

- 📍 **Géolocalisation**: Trouvez les stations les plus proches
- 💰 **Comparaison de Prix**: Comparez les prix de différents carburants
- 🔔 **Alertes de Prix**: Recevez des notifications pour les meilleurs prix
- 🗺️ **Visualisation Cartographique**: Visualisez les stations sur une carte
- 📊 **Statistiques**: Analysez les tendances des prix
- 📱 **Progressive Web App**: Utilisable hors-ligne et installable

## 🏗️ Architecture

- **Frontend**: React 18 + Vite
- **Cartographie**: Leaflet
- **PWA**: vite-plugin-pwa
- **Styling**: CSS pur avec variables personnalisées
- **API**: data.economie.gouv.fr

## 📦 Installation

```bash
npm install
```

## 🚀 Développement

```bash
npm run dev
```

L'application sera disponible sur `http://localhost:5173`

## 🔨 Build

```bash
npm run build
```

## 📱 Utilisation

### Navigation
- **Liste**: Visualisez toutes les stations avec leurs prix
- **Carte**: Explorez les stations sur une carte interactive
- **Stats**: Consultez les statistiques des prix

### Alertes
- Configurez des alertes pour être notifié quand les prix baissent
- Sélectionnez une station, un carburant et un prix seuil
- Recevez des notifications en temps réel

### Géolocalisation
- L'app détecte automatiquement votre position
- Ajustez le rayon de recherche (5-50 km)
- Possibilité de chercher par adresse

## 🎨 Design

L'application utilise un design moderne avec:
- Thème sombre par défaut
- Couleurs personnalisées par type de carburant
- Animations fluides et transitions
- Interface responsive et tactile
